38 /* Included:
39  * Matrix Convolver
40  * y = H * x; looped/summed over in/output channels, applied block-by-block
41  * where
42  * y: nOutputChannels x blockSize
43  * x: nInputChannels x blockSize
44  * H: nOutputChannels x nInputChannels x filterLength
45  *
46  * Multi Convolver
47  * y = H * x; looped over channels, applied block-by-block
48  * where
49  * y: nChannels x blockSize
50  * x: nChannels x blockSize
51  * H: nChannels x filterLength
52  */
